Le **suivi conversions Carrd** te permet de voir exactement quelles campagnes génèrent des conversions sur ta landing page Carrd. Le dashboard Pimms inclut un **assistant de configuration interactif** qui gère tous les détails techniques pour toi.

Tu obtiendras des **analytics conversion en temps réel** montrant quelles publicités, posts sociaux ou campagnes email génèrent de vraies inscriptions, réservations ou achats depuis ton site Carrd. Parfait pour le **suivi link-in-bio** et l'**attribution landing page** du clic à la conversion.

Carrd est l'outil de référence pour créer des sites one-page simples et beaux — largement utilisé par les indie hackers, créateurs et startups. Avec Pimms, tu peux suivre chaque conversion sur ton site Carrd et savoir exactement d'où vient chaque lead.

## Pourquoi suivre les conversions Carrd avec Pimms ?

- **Sache quelles campagnes génèrent des inscriptions.** Tu partages ta landing page Carrd sur Twitter, dans ta newsletter et via Product Hunt. Sans Pimms, tu vois des soumissions mais tu ne sais pas quelle source les a générées. Avec Pimms, chaque conversion est attribuée à la campagne exacte.

- **Mesure les performances de ton link-in-bio.** Tu utilises Carrd comme page link-in-bio avec des liens vers ton checkout Stripe, ta réservation Cal.com et ton inscription newsletter. Pimms suit quelle plateforme sociale génère les clics et conversions les plus précieux.

- **Teste des variantes de landing page.** Tu crées plusieurs pages Carrd pour différentes audiences. Crée des liens Pimms séparés pour chacune et vois quelle page génère le meilleur taux de conversion par source de trafic.

## Prérequis

- Un **plan Carrd Pro Plus** ou supérieur (requis pour l'accès au code personnalisé)
- Un **compte Pimms** ([inscris-toi ici](https://app.pimms.io))
- Un site Carrd avec au moins un formulaire ou lien sortant

## Ce que tu vas suivre

- **Soumissions formulaires** — Chaque soumission de formulaire Carrd avec les détails du visiteur et la campagne qui l'a amené
- **Clics liens sortants** — Suis quand les visiteurs cliquent vers Cal.com, Stripe, Tally ou d'autres services
- **Attribution conversion** — Quelle publicité, email, post social ou lien bio a généré chaque action
- **Performance landing page** — Compare comment les différentes sources de trafic convertissent sur ton site Carrd

## Comment configurer

Rends-toi sur ton [dashboard Pimms](https://app.pimms.io), ouvre **Intégrations → Carrd** et lance l'assistant de configuration. Il te guide pour ajouter le script de détection sur ton site Carrd, configurer le suivi des domaines sortants et vérifier avec une visite test. La configuration prend environ 3 minutes.

## Ce que l'assistant de configuration couvre

L'assistant Carrd a 5 étapes :

1. **Installer le script de détection** — Ajoute le script Pimms dans la section head de ton site Carrd
2. **Vérifier l'installation du script** — Confirme que le script se charge sur ton site Carrd publié
3. **Configurer le suivi sortant** — Configure `data-domains` pour suivre les clics vers Cal.com, Stripe, Tally et d'autres services
4. **Créer un lien test** — Génère un lien de suivi vers ta page Carrd
5. **Vérifier le fonctionnement** — Clique et effectue une conversion test pour confirmer que l'attribution est connectée

## Comment ça marche

1. **Ajoute le script de détection Pimms** dans la section head de ton site Carrd via Settings > Head (nécessite le plan Pro Plus)
2. **Le script détecte les visiteurs** arrivant depuis des liens Pimms et stocke les données d'attribution
3. **Quand le visiteur convertit** — remplit un formulaire, clique vers Cal.com, complète un paiement Stripe — les données d'attribution suivent
4. **Pimms attribue la conversion** à la campagne d'origine, visible instantanément dans ton dashboard

Pour les liens sortants vers les services tiers, le script de détection ajoute automatiquement l'identifiant de suivi aux domaines supportés configurés dans `data-domains`.

## Meilleures façons de partager tes liens de suivi

- **Bio Instagram** — Remplace ton URL Carrd brute par un lien Pimms pour suivre combien de conversions viennent d'Instagram
- **Profil Twitter/X** — Utilise un lien Pimms tracké dans ta bio pour mesurer les conversions générées par Twitter
- **Signatures email** — Ajoute un lien Pimms vers ta page Carrd dans ta signature email
- **Lancements Product Hunt** — Suis le trafic et les conversions du jour de lancement avec un lien Pimms dédié
- **Publicités payantes** — Pointe tes pubs vers un lien Pimms de ta landing page Carrd pour mesurer le ROI publicitaire

## Ce que tu verras dans ton dashboard

Ton dashboard Pimms affiche chaque conversion de ton site Carrd en temps réel avec l'attribution complète. Pour chaque action — soumission formulaire, réservation ou paiement — tu verras quel lien Pimms ils ont cliqué, depuis quelle plateforme, et quelle campagne. Filtre par date ou source de trafic pour comprendre quels canaux convertissent le mieux sur ta page Carrd.

## FAQs

::: faq

### Ai-je besoin d'un plan Carrd payant pour le suivi ?

Oui, tu as besoin d'un plan Carrd Pro Plus ou supérieur pour ajouter du code personnalisé (le script de détection Pimms) à ton site. Le plan Pro standard n'inclut pas les capacités de code personnalisé nécessaires pour les scripts de suivi.

:::

::: faq

### Puis-je suivre les soumissions de formulaires et les clics de liens sortants ?

Oui ! Le script de détection Pimms suit automatiquement les clics de liens sortants vers les services tiers (Cal.com, Stripe, Tally) quand tu configures l'attribut `data-domains`. Les soumissions de formulaires via code personnalisé sont aussi suivies via l'approche champ caché.

:::

::: faq

### Est-ce que ça marche avec le gestionnaire de formulaires intégré de Carrd ?

Le gestionnaire de formulaires intégré de Carrd fonctionne avec le script de détection pour l'attribution de base. Pour un suivi complet au niveau webhook, tu peux utiliser un embed Tally ou Jotform sur ta page Carrd, qui ont tous les deux leurs propres intégrations Pimms — voir le [guide Tally](/articles/how-to-track-tally-webhook) ou le [guide Jotform](/articles/how-to-track-jotform).

:::

::: faq

### Puis-je suivre plusieurs sites Carrd ?

Oui ! Installe le script de détection Pimms sur chaque site Carrd et crée des liens Pimms séparés pour chacun. Toutes les conversions de tous tes sites Carrd apparaîtront dans ton dashboard Pimms avec l'attribution complète.

:::

::: faq

### Puis-je utiliser le suivi Carrd avec Stripe ou Cal.com ?

Oui. Si tu lies un checkout Stripe ou Cal.com depuis ta page Carrd, le script de détection transmet l'identifiant de suivi à ces services. Configure l'assistant de chaque intégration dans le dashboard et ajoute les domaines dans `data-domains`.

:::
