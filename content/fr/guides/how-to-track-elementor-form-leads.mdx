Le **tracking des formulaires WordPress Elementor** te permet de voir exactement quelles campagnes marketing génèrent tes meilleurs leads. Le tableau de bord Pimms inclut un **assistant de configuration interactif** qui gère tous les détails techniques pour toi.

Tu obtiendras une **attribution des leads en temps réel** montrant quelles pubs, posts YouTube ou campagnes email génèrent de vraies soumissions de formulaires. Parfait pour le **suivi des leads WordPress** avec une **attribution marketing** complète sur tous tes formulaires Elementor Pro.

WordPress alimente des millions de sites business, et Elementor Pro est l'un des page builders les plus populaires pour créer des formulaires de génération de leads. Avec Pimms, tu peux suivre chaque soumission de formulaire et l'attribuer exactement à la campagne qui a amené le visiteur.

## Pourquoi suivre les formulaires Elementor avec Pimms ?

- **Sache quelles pubs génèrent vraiment des leads, pas seulement des clics.** Tu dépenses 2 000 $/mois en Google Ads pour diriger le trafic vers ton site WordPress. Sans Pimms, tu vois les clics pub et les soumissions de formulaires séparément. Avec Pimms, chaque soumission de formulaire est liée à la pub et au mot-clé spécifiques qui ont amené le visiteur.

- **Mesure tes efforts SEO en leads réels.** Ton blog WordPress se classe pour plusieurs mots-clés. Les visiteurs lisent les articles et remplissent les formulaires de contact. Pimms relie chaque soumission à la page et à la source de trafic d'origine, te montrant quel contenu génère de vrais leads.

- **Compare les performances des landing pages.** Tu as créé plusieurs landing pages Elementor pour différentes campagnes. Pimms te montre quelle page génère le plus de soumissions de formulaires et depuis quelle source de trafic — pour optimiser ton funnel.

## Comment le configurer

Va sur ton [tableau de bord Pimms](https://app.pimms.io), ouvre **Integrations → WordPress / Elementor**, et lance l'assistant de configuration. Il te guide pour ajouter deux scripts dans le head de ton site WordPress, créer un formulaire Elementor compatible, configurer l'action webhook et vérifier avec une soumission test. La configuration prend environ 5 minutes.

## Ce que couvre l'assistant de configuration

L'assistant WordPress / Elementor comprend 6 étapes :

1. **Installer les scripts de tracking** — Script de détection et script d'injection de formulaire ajoutés au head de ton WordPress
2. **Vérifier l'installation des scripts** — Confirme que les deux scripts se chargent correctement sur ton site
3. **Créer un formulaire compatible Pimms** — Suis les exigences des champs pour les formulaires Elementor
4. **Configurer le webhook Elementor** — Ajoute l'action webhook dans "Actions After Submit" de ton formulaire Elementor
5. **Créer un lien test** — Génère un lien de tracking vers ta page WordPress
6. **Vérifier que le tracking fonctionne** — Soumets un formulaire test pour confirmer que tout est connecté

## Comment ça fonctionne

1. **Installe les scripts Pimms** sur ton site WordPress — le script de détection identifie les visiteurs venant des liens Pimms, et le script d'injection de formulaire ajoute un champ de tracking caché à tes formulaires Elementor
2. **Crée un lien Pimms** pointant vers ta page WordPress (ex. `pim.ms/free-consultation`)
3. **Un visiteur clique sur ton lien**, arrive sur ton site WordPress et remplit un formulaire Elementor
4. **Elementor déclenche un webhook** vers Pimms avec les données du formulaire et l'ID de tracking, et Pimms attribue le lead à la campagne d'origine

Le script d'injection de formulaire fonctionne automatiquement avec les formulaires Elementor Pro — il ajoute un champ caché `pimms_id` qui capture les données d'attribution du visiteur.

- **Posts LinkedIn** — Partage ta landing page via Pimms pour suivre les leads venant des réseaux sociaux
- **Guest blogging** — Inclus des liens trackés dans tes guest posts pour mesurer la génération de leads par referral

## FAQs

::: faq

### Combien de temps prend la configuration du tracking des formulaires WordPress Elementor ?

La configuration complète du tracking des formulaires WordPress Elementor prend quelques minutes grâce à l'assistant interactif du tableau de bord Pimms. L'assistant gère l'installation des scripts, la configuration du webhook et la vérification. La plupart des utilisateurs terminent la configuration en une session avec des résultats immédiats.

:::

::: faq

### Le tracking des formulaires Elementor fonctionne-t-il avec la version gratuite ?

Non, le tracking des formulaires WordPress Elementor nécessite **Elementor Pro** pour la fonctionnalité webhook et les fonctionnalités avancées des formulaires. La version gratuite d'Elementor n'inclut pas les widgets de formulaire ni les capacités webhook nécessaires pour le suivi des conversions et l'attribution des leads.

:::

::: faq

### Je peux suivre plusieurs formulaires Elementor sur un seul site WordPress ?

Oui ! Une fois que tu as configuré le tracking Pimms, les formulaires Elementor sur tout ton site WordPress sont trackables. Chaque soumission de formulaire est attribuée à la campagne, pub ou canal spécifique qui a amené le visiteur, pour une visibilité complète sur les analytics de conversion.

:::

::: faq

### Ça fonctionne avec d'autres plugins de formulaire WordPress ?

L'assistant de configuration est conçu spécifiquement pour les formulaires Elementor Pro. Si tu utilises d'autres plugins de formulaire (Contact Form 7, WPForms, Gravity Forms), le script de détection fonctionnera toujours, mais tu devras configurer l'action webhook différemment. Contacte le support pour de l'aide avec les plugins de formulaire non-Elementor.

:::
