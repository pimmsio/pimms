L'**intégration Cal.com + Webflow** te permet d'intégrer Cal.com directement sur ton site Webflow avec une attribution complète des réservations. Le dashboard Pimms inclut des assistants de configuration interactifs pour Cal.com et Webflow — deux assistants rapides et tu suis les réunions de bout en bout.

C'est la configuration idéale pour les agences, consultants et entreprises de services qui utilisent Webflow pour construire leur site marketing et Cal.com pour planifier les appels clients. Tu verras exactement quelles campagnes génèrent de vraies réunions réservées via tes pages Webflow.

Webflow est idéal pour construire des sites marketing professionnels avec des widgets de réservation intégrés. Pimms ajoute la pièce manquante : **l'attribution**. Tu sais maintenant quelle page, campagne ou pub a amené chaque prospect qui a réservé une réunion.

## Pourquoi suivre les réservations Cal.com sur ton site Webflow ?

- **Sache quelle page de service génère le plus de consultations.** Tu as des pages Webflow séparées pour chaque service que tu proposes, chacune avec un widget de réservation Cal.com intégré. Pimms te montre quelles pages génèrent le plus de réservations et depuis quelle source de trafic.

- **Prouve le ROI à tes clients.** Si tu es une agence qui mène des campagnes générant du trafic vers un site Webflow avec réservation Cal.com, Pimms te donne des données claires à montrer aux clients : « Ta campagne LinkedIn a généré 14 réservations de démo ce mois-ci. »

- **Compare le SEO vs le trafic payant pour la génération de réunions.** Ton site Webflow se positionne bien en organique et tu lances aussi des pubs payantes. Les deux génèrent du trafic vers ta page de réservation. Pimms te montre quelle source génère plus de réunions — tu sais ainsi où investir.

## Prérequis

- Un **compte Pimms** ([inscription ici](https://app.pimms.io))

## Ce que tu vas suivre

- **Les réservations Cal.com depuis ton site Webflow** — Chaque réunion réservée via ton widget Cal.com intégré
- **L'attribution des réunions par campagne** — Quelle pub, email, post social ou source de référence a généré chaque réservation
- **La performance au niveau des pages** — Vois quelles pages Webflow génèrent le plus de réunions

## Comment configurer

Cette intégration combine deux assistants de configuration dans ton [dashboard Pimms](https://app.pimms.io) :

1. **Termine d'abord l'assistant Cal.com** — Cela configure le webhook qui envoie les événements de réservation à Pimms et ajoute le champ de tracking à tes formulaires Cal.com
2. **Puis termine l'assistant Webflow** — Cela ajoute le script de détection à ton site Webflow pour identifier les visiteurs venant des liens Pimms

Les deux assistants guident étape par étape. Le temps de configuration total est d'environ 10 minutes pour les deux.

## Ce que couvrent les assistants de configuration

**Assistant Cal.com (6 étapes) :**
1. Installer le script de détection sur ton site Webflow
2. Vérifier l'installation du script
3. Ajouter le champ de tracking pimms_id aux questions de réservation Cal.com
4. Configurer le webhook Cal.com
5. Créer un lien de test
6. Vérifier que le tracking fonctionne

**Assistant Webflow (6 étapes) :**
1. Installer les scripts de tracking (détection + injection de formulaire)
2. Vérifier l'installation des scripts
3. Construire un formulaire compatible Pimms (si applicable)
4. Configurer le webhook Webflow (pour le tracking des formulaires)
5. Créer un lien de test
6. Vérifier que le tracking fonctionne

Si tu as seulement besoin du suivi des réservations Cal.com (pas de formulaires Webflow), l'assistant Cal.com seul suffit.

## Comment ça fonctionne

1. **Le script de détection Pimms** est installé sur ton site Webflow — quand un visiteur arrive via un lien Pimms, le script détecte l'ID de clic
2. **Le script injecte automatiquement l'ID de tracking** dans les intégrations Cal.com de la page via le champ caché `pimms_id`
3. **Le visiteur réserve une réunion** via ton widget Cal.com intégré — la réservation inclut l'ID de clic Pimms
4. **Cal.com déclenche un webhook** vers Pimms avec les données de réservation et l'ID de tracking
5. **Pimms attribue la réunion** à la campagne d'origine, visible dans ton dashboard

Le script de détection gère automatiquement la passation entre Webflow et Cal.com — pas besoin de passer manuellement les paramètres d'URL.

## FAQs

::: faq

### Ai-je besoin d'un plan Webflow payant pour l'intégration Cal.com ?

Ça dépend de ta configuration. Le plan gratuit Webflow autorise du code personnalisé limité, mais un plan payant donne un accès complet à l'injection de code head et aux scripts site-wide. La plupart des utilisateurs ont un plan Webflow payant et peuvent ajouter le script de tracking facilement.

:::

::: faq

### Puis-je personnaliser l'apparence du formulaire de réservation Cal.com ?

Oui ! Cal.com propose des options de personnalisation étendues pour les couleurs, la marque et les champs du formulaire. Tu peux les personnaliser dans ton dashboard Cal.com sous Event Types → Advanced → Booking Questions.

:::

::: faq

### Est-ce que ça fonctionne avec le CMS Webflow ?

Oui ! Tu peux utiliser le CMS Webflow pour créer des pages de réservation dynamiques avec différents liens Cal.com pour différents membres d'équipe ou services. Le script de détection fonctionne sur toutes les pages, y compris les pages de collections CMS.

:::

::: faq

### Puis-je utiliser les liens courts Pimms pour la méthode calendrier plein écran au lieu d'ajouter ?pimms_id=1 ?

Oui, mais cette approche casse le tracking d'attribution. Quand tu utilises un lien court Pimms directement vers Cal.com, toutes les réservations seront attribuées au clic sur le bouton plutôt qu'à la vraie source de trafic.

**Utilise les liens courts Pimms uniquement pour :**
- Des tests A/B à court terme (max 1 mois)
- Tester quel design de bouton convertit le mieux
- Une configuration rapide quand tu as besoin de tracking immédiat

Pour un suivi à long terme et une attribution correcte, utilise toujours le lien Cal.com avec le paramètre `?pimms_id=1`.

:::

::: faq

### Puis-je suivre à la fois les soumissions de formulaires Webflow ET les réservations Cal.com ?

Oui ! Si tu as à la fois des formulaires Webflow et des widgets de réservation Cal.com sur ton site, configurer les deux assistants permettra de suivre toutes les conversions. Les soumissions de formulaires et les réservations de réunions apparaîtront toutes deux dans ton dashboard Pimms avec une attribution complète.

:::
