Suis chaque paiement Stripe avec une attribution de vente complète et des analytics de conversion e-commerce.

Vois exactement quelles campagnes génèrent tes meilleurs clients avec **Pimms** — obtiens une attribution de paiement complète dans ton tableau de bord.

Pimms est officiellement listé sur la marketplace Stripe :  
👉 [Voir la fiche](https://marketplace.stripe.com/apps/pimms)

Tu as deux façons de connecter Stripe à Pimms :

- **Option A :** Partager des liens courts traqués créés avec Pimms (idéal pour les emails, LinkedIn, réseaux sociaux, etc.)
- **Option B :** Ajouter le tracking Pimms sur ton site tout en gardant les boutons Stripe tels quels

Allons-y étape par étape.

## 1. Connecter Stripe à Pimms

1. Connecte-toi à ton [dashboard Pimms](https://app.pimms.io)  
2. Va dans **Intégrations → Stripe** et clique sur **Activer**  
3. Choisis **mode Test** ou **Production**  
4. Clique sur **Accepter** pour autoriser l’accès à Stripe

![Autoriser Stripe](https://assets.pimms.io/stripe-accept-permissions.webp)

5. Vérifie tes paramètres Stripe :
   - [Paramètres production](https://dashboard.stripe.com/settings/apps/pimms.io)
   - [Paramètres test](https://dashboard.stripe.com/test/settings/apps/pimms.io)

Si besoin :
- Va dans **Paramètres → Équipe et sécurité → Applications**
- Trouve **Pimms** et clique sur **Connecter l’espace de travail**

![Connexion Stripe](https://assets.pimms.io/stripe-connect-workspace.webp)

- Choisis ton workspace et confirme

![Autoriser Pimms Stripe](https://assets.pimms.io/stripe-authorize-permissions.webp)

✅ C’est connecté !

![Connexion réussie](https://assets.pimms.io/stripe-connected.webp)

## 2. Créer ton lien de paiement Stripe

1. Dans ton [dashboard Stripe](https://dashboard.stripe.com), clique sur **Créer un lien de paiement**  
2. Configure ton produit et ton tarif  
3. Copie le lien de paiement généré (ex. `https://buy.stripe.com/xyz`)

![Créer lien de paiement Stripe](https://assets.pimms.io/stripe-create-payment-link.webp)

## 3. Partager ton lien de paiement et suivre les ventes (2 options)

Tu as deux façons principales de suivre les paiements Stripe avec Pimms — selon la manière dont le lien de paiement est utilisé.

### ✅ Option A : Utiliser un lien court Pimms qui pointe directement vers Stripe

C’est la **façon la plus rapide et la plus simple de commencer**.

À utiliser lorsque tu partages un **lien de paiement Stripe directement**, sans passer par une autre page ou un site intermédiaire.

1. Va sur [Pimms](https://app.pimms.io)  
2. Crée un lien court et colle ton lien de paiement Stripe  
3. Active le **suivi des conversions**

Tu obtiendras un lien comme :

```
https://pim.ms/xyz
```

Quand quelqu’un clique sur ce lien et paie, Pimms attribue automatiquement la vente au lien d’origine — avec toutes les statistiques qui vont avec.

---

### ✅ Option B : Déclencher le paiement Stripe depuis ton site web

À utiliser si le lien de paiement Stripe est **déclenché par un clic sur ton propre site** (par exemple via un bouton "Acheter").

Point essentiel :  
**Ne colle pas un lien court Pimms directement dans le HTML de ton site.**

Pourquoi ?  
Parce que toutes les ventes seront attribuées à ce même lien intégré — **et non au lien Pimms d’origine** qui a amené le visiteur sur ton site.

👉 Ce que tu veux, c’est attribuer la vente au **lien Pimms qui a été cliqué en premier** (email, pub, post, etc.)

Voici comment faire :

1. Utilise ton lien Stripe normal et **ajoute ce paramètre** :

```html
<a href="https://buy.stripe.com/xyz?pimms_client_reference_id=1">Acheter maintenant</a>
```

2. Intègre le script de tracking Pimms dans le HTML global de ton site (dans la balise `<head>`) :

Important: Injecte le script suivant dans tous les pages de ton site.

```html
<script defer src="https://cdn.pimms.io/analytics/script.detection.js"></script>
```

Pour les développeurs : alternativement, utilise le [SDK javascript et le tracking serveur](/articles/introducing-conversion)

### ✅ Option C: Déclencher le paiement Stripe depuis ton site (Avec Stripe Checkout)

Pour cette option, veuillez consulter ce guide spécifique : [Guide pour suivre les ventes avec Stripe Checkout](/articles/how-to-track-sales-with-stripe-checkout).

Une fois ce setup en place — c’est bon ✅

Tu peux créer **autant de liens courts Pimms que tu veux** qui pointent vers ton site.  
Chaque lien attribuera automatiquement la vente à la bonne source, selon le clic d’origine du visiteur.

## 4. Voir tes ventes Stripe dans Pimms

Chaque vente traquée apparaîtra dans ton [dashboard Pimms](https://app.pimms.io), avec attribution complète :

- Source (ex. LinkedIn, Email, Ads)
- Canal
- Campagne
- Lien cliqué

![Suivi des ventes avec Pimms](https://assets.pimms.io/stripe-sales-tracking.webp?v=2)

## FAQ

::: faq

### Est-ce que ça fonctionne avec les Payment Links et Checkout Stripe ?

Oui ! Ce guide couvre les Stripe Payment Links. Pour Stripe Checkout (l'option intégrée), consulte notre [guide dédié Stripe Checkout](/articles/how-to-track-sales-with-stripe-checkout) pour développeurs.

:::

::: faq

### Puis-je suivre les paiements d'abonnement et la facturation récurrente ?

Oui ! Les paiements uniques et les inscriptions d'abonnement sont suivis. Pour les abonnements, tu verras la conversion initiale quand les clients s'abonnent pour la première fois, plus l'attribution du revenu récurrent.

:::

::: faq

### Et si j'ai plusieurs produits Stripe ou niveaux de prix ?

Tous tes produits Stripe seront suivis automatiquement une fois connectés. Chaque vente affichera le produit spécifique, le montant et les données d'attribution dans ton tableau de bord Pimms.

:::
