::: faq
### Combien de temps faut-il pour configurer une intégration ?
La plupart des intégrations (Stripe, Cal.com, Tally) prennent moins de 5 minutes. Connecte simplement ton compte et PIMMS commence à tracker automatiquement.
:::

::: faq
### Le tracking Stripe est-il vraiment gratuit ?
Oui. Le tracking de conversions Stripe et Shopify est inclus dans le plan gratuit. Aucune carte bancaire requise.
:::

::: faq
### Puis-je utiliser Zapier pour connecter PIMMS à mon CRM ?
Oui. PIMMS est disponible sur le marketplace Zapier. Pousse les événements de clics, leads et ventes vers HubSpot, Salesforce, Notion, Airtable, Slack et 5 000+ apps.
:::

::: faq
### Dois-je installer un SDK ?
Non. La plupart des intégrations fonctionnent via webhooks ou connexions natives — aucun code requis. Pour les cas avancés, nous fournissons un SDK TypeScript et une API REST.
:::

::: faq
### Quels outils d'analytics fonctionnent avec PIMMS ?
PIMMS fonctionne en parallèle avec Google Analytics, Mixpanel, Amplitude et tout outil qui lit les paramètres UTM. Utilise les deux pour des insights complémentaires.
:::
