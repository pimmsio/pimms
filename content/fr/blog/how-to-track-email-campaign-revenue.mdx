### Comment fonctionnent les paramètres UTM

Les paramètres UTM sont des balises ajoutées à tes liens. Ils permettent d’identifier l’origine du trafic et de l’associer à des conversions. Ce sont comme des miettes de pain numériques :

- **Source** : d’où vient le trafic (ex. : email)
- **Medium** : méthode utilisée (ex. : newsletter, promo)
- **Campagne** : nom de la campagne
- **Contenu** : distingue plusieurs liens dans un même email

PIMMS les interprète parfaitement, ce qui te donne une vision fine de ce qui fonctionne.

## Mesurer le succès de ta campagne email

### Indicateurs clés à suivre

Voici les KPIs que tu dois surveiller :

- **RPE (revenu par email)** : revenu total / emails envoyés
- **Panier moyen (AOV)** : revenu total / nombre de commandes
- **ROI** : (revenu - coût) / coût
- **Taux de conversion** : % de destinataires ayant acheté

PIMMS calcule tout ça en direct via son tableau de bord. Avec l’offre Pro, tu peux suivre jusqu’à 20 000 $ de ventes par mois.

### Analyse de tes données

Décompose les résultats pour trouver les tendances :

**Par appareil**

- Compare les conversions mobile vs desktop
- Suis les ventes générées par l’app mobile
- Observe les comportements selon les systèmes d’exploitation

**Par zone géographique**

- Analyse les revenus par région
- Identifie tes meilleurs marchés
- Ajuste tes campagnes selon les fuseaux horaires

Exemple : le taux de conversion peut aller de 76 % sur LinkedIn à 1,6 % sur Google [\[1\]](https://pimms.io)

### Vérifie la fiabilité de tes données

1\. **Validation**

- Compare les données PIMMS à celles de Shopify ou Stripe

- Repère les anomalies

2\. **Intégration technique**

## Astuces pour améliorer ton tracking

### Bonnes pratiques pour nommer les UTM

Sois cohérent pour une donnée exploitable :

**utm_source**

- En minuscules, sans espace
- Descripteur clair : `newsletter`, `promo`
- Ajoute le nom de la plateforme si besoin

**utm_medium**

- Toujours "email" (évite "e-mail" ou "mail")

**utm_campaign**

- Tirets à la place des espaces
- Date au format `2025-05`
- Type de campagne : promo, welcome, nurture

Exemple :  
`?utm_source=newsletter&utm_medium=email&utm_campaign=2025-05-spring-sale`

### Gérer les problèmes d’attribution

- **Multi-appareil** : vérifie que le suivi fonctionne partout
- **Multiples clics** : garde la source d’origine et attribue correctement la vente

### Tester tes liens

**Avant le lancement** :

- Teste sur iOS, Android, desktop
- Vérifie les liens transférés
- Ouvre bien dans les apps
- Vérifie la présence des UTMs

**Après lancement** :

- Utilise le dashboard PIMMS pour :
  - Suivre les clics
  - Analyser les conversions
  - Corriger les erreurs rapidement
