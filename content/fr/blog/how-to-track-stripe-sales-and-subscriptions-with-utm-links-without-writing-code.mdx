**Tu veux savoir exactement quelles actions marketing génèrent des ventes sur [Stripe](https://stripe.com/) ?** Le suivi UTM permet de lier facilement les transactions Stripe à des campagnes précises – sans écrire de code. Voici comment faire :

- **C’est quoi un lien UTM ?** Ce sont des balises ajoutées à une URL (ex. `?utm_source=facebook`) qui indiquent d’où vient ton trafic et comment il convertit.
- **Pourquoi les utiliser avec Stripe ?** Pour attribuer tes revenus aux pubs, e-mails ou autres canaux, directement dans Stripe.
- **Comment les configurer ?** Utilise les Payment Links de Stripe, ajoute des paramètres UTM, et collecte les données automatiquement avec des outils comme [Zapier](https://zapier.com/) ou [Make](https://www.make.com/en).
- **Des outils pour simplifier ?** Des plateformes comme [PIMMS](https://pimms.io/) [capturent automatiquement les données UTM](/articles/utm-guide) et offrent des analytics en temps réel pour améliorer tes campagnes.

**Exemple d’URL avec UTM :**

```
https://buy.stripe.com/test?utm_source=google&utm_medium=cpc&utm_campaign=summer_sale
```

## Paramètres UTM : les éléments clés pour l’attribution des ventes

Les paramètres UTM sont essentiels pour suivre les revenus Stripe. Ils révèlent l’origine de tes clients via cinq éléments principaux [\[2\]](https://www.spinutech.com/digital-marketing/analytics/conversion/what-are-utm-codes-utm-tracking-explained).

### Structure de base des paramètres UTM

Ces cinq paramètres UTM offrent des insights détaillés sur tes sources de trafic :

| Paramètre        | Rôle                                      | Exemple                        |
| ---------------- | ------------------------------------------ | ------------------------------ |
| **utm_source**   | Source du trafic                          | google, facebook, newsletter   |
| **utm_medium**   | Canal marketing                           | cpc, email, social             |
| **utm_campaign** | Nom de la campagne                        | summer-sale-2025               |
| **utm_content**  | Différenciation entre liens similaires    | orange-cta, header-link        |
| **utm_term**     | Mot-clé spécifique (search)              | camping-gear                   |

> "Le code UTM ou ‘Urchin Tracking Module’ est une chaîne ajoutée à la fin d'une URL pour suivre l’origine du trafic vers ton site. Elle détaille la source, le canal, la campagne, voire même le mot-clé." – John Abbasi, Peel [\[4\]](https://www.peelinsights.com/post/what-is-a-utm-tracking-code)

Une fois cette structure en place, il faut intégrer les données UTM dans Stripe pour une attribution précise.

### Connexion des données UTM aux transactions [Stripe](https://stripe.com/)

![Stripe](https://assets.seobotai.com/pimms.io/682b6c814fa53d42207bccae/bd96116015edf6c6fa9e8dbfe6e54dc1.jpg)

Pour assurer un suivi efficace :

- Utilise uniquement des caractères alphanumériques, tirets ou underscores dans les codes UTM [\[1\]](https://docs.stripe.com/payment-links/url-parameters).
- Garde-les sous 150 caractères [\[1\]](https://docs.stripe.com/payment-links/url-parameters).
- Configure Stripe pour rediriger après le paiement [\[1\]](https://docs.stripe.com/payment-links/url-parameters).

Bien intégrés, les paramètres UTM boostent ton [tracking de campagnes](/articles/introducing-conversion). En février 2023, un client d’[Attributer](https://attributer.io/) est passé de 60 % à 95 % de précision d’attribution grâce aux données UTM dans Stripe [\[3\]](https://mailchimp.com/resources/utm-links).

Autre exemple :

```
www.yoursite.com/pricing?utm_source=facebook&utm_medium=paid-social&utm_campaign=spring2025&utm_content=video-ad
```

⚠️ Utilise toujours des lettres **minuscules** dans les UTM [\[4\]](https://www.peelinsights.com/post/what-is-a-utm-tracking-code).

## Guide no-code pour configurer le suivi UTM dans Stripe

Voici comment configurer le tracking UTM dans Stripe **sans coder**.

### Ajouter des balises UTM aux Payment Links Stripe

Tu peux configurer cela directement depuis le tableau de bord Stripe :

- **Accède aux paramètres de ton Payment Link** : Connecte-toi à Stripe et sélectionne un lien. Clique sur la flèche à côté de "Copier", puis sur "Paramètres URL" [\[1\]](https://docs.stripe.com/payment-links/url-parameters).
- **Ajoute les paramètres UTM** : Ajoute-les à ton lien :

```
https://buy.stripe.com/test_eVa5nPg1j1wmfXq5kr?utm_medium=earned_email&utm_source=marketo&utm_campaign=campaign_a
```

Pense à automatiser ce processus pour garder une cohérence sur toutes tes campagnes.

### Automatiser le suivi UTM avec [Zapier](https://zapier.com/) et [Make](https://www.make.com/en)

![Zapier](https://assets.seobotai.com/pimms.io/682b6c814fa53d42207bccae/17c1a8b06dbdeb853898ddf955aeea3f.jpg)

Inclue le paramètre `client_reference_id` dans ta config Stripe pour fiabiliser l’attribution.

## Guide de maintenance du tracking UTM

Une fois en place, tu dois maintenir la qualité de tes données.

### Vérifications régulières de qualité des liens UTM

| **Problème courant**     | **Impact**                                | **Solution recommandée**                      |
| ------------------------ | ----------------------------------------- | --------------------------------------------- |
| Majuscules dans les UTM  | Données fragmentées                       | Toujours en minuscules                        |
| Noms incohérents         | Campagnes mal regroupées                  | Créer une nomenclature claire et documentée   |
| Fautes de frappe         | Données orphelines                        | Audit régulier de tes liens                   |
| Espaces encodés          | Erreurs de tracking                       | Utiliser `%20` ou des tirets                  |

Une fois standardisé, vérifie la cohérence dans PIMMS pour fiabiliser ton attribution.
