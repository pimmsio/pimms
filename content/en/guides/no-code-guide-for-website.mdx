## FAQs

::: faq

### Which no-code platforms support Pimms tracking?

Pimms works with any no-code platform that allows custom code in the head section, including Webflow, Framer, Bubble, Wix, and most AI website builders. The platform just needs to support adding JavaScript.

:::

::: faq

### Do I need separate tracking for each tool I use?

No! Once you install the Pimms script on your no-code site, it automatically works with all supported tools (Cal.com, Tally, Stripe, etc.) without additional setup for each tool.

:::

::: faq

### Can I track conversions across different pages and funnels?

Yes! Pimms tracks visitors across your entire no-code site. Whether someone fills a form on page 1 and books a meeting on page 3, you'll see the complete conversion path with full attribution.
