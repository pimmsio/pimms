**Cal.com + Framer integration** lets you embed Cal.com directly into your Framer site with complete booking attribution. The Pimms dashboard includes interactive setup wizards that handle both Cal.com webhooks and Framer script installation — two quick wizards and you're tracking meetings end-to-end.

This is the ideal setup for founders and teams who use Framer to build their website and Cal.com to schedule demos, consultations, or sales calls. You'll see exactly which campaigns drive actual booked meetings through your Framer site.

Framer makes it easy to embed Cal.com booking widgets. Pimms adds the missing piece: **attribution**. Now you know not just how many meetings were booked, but which LinkedIn post, ad, or email campaign brought each prospect.

## Why track Cal.com bookings on your Framer site?

- **Attribute demo bookings to specific campaigns.** You drive traffic to your Framer site from multiple sources — LinkedIn, Product Hunt, Google Ads. Without Pimms, you see bookings in Cal.com but don't know which source generated them. With Pimms, every booking shows the exact campaign that brought the visitor.

- **Measure your landing page effectiveness.** You built multiple Framer landing pages for different audiences (agencies, startups, enterprise). Each page has an embedded Cal.com widget. Pimms shows you which page generates the most meetings from each traffic source.

- **Track the full funnel from click to booked meeting.** A visitor clicks your Pimms link, lands on your Framer site, browses your features page, and books a call. Pimms tracks the entire journey — so you know which first touchpoint started the path to a meeting.

## How to set it up

This integration combines two setup wizards in your [Pimms dashboard](https://app.pimms.io):

1. **Complete the Cal.com wizard first** — This configures the webhook that sends booking events to Pimms and adds the tracking field to your Cal.com forms
2. **Then complete the Framer wizard** — This adds the detection script to your Framer site so visitors from Pimms links are identified

Both wizards are guided step-by-step. Total setup time is about 10 minutes for both.

## What the setup wizards cover

**Cal.com wizard (6 steps):**
1. Install the detection script on your Framer site
2. Verify script installation
3. Add the pimms_id tracking field to Cal.com booking questions
4. Configure the Cal.com webhook
5. Create a test link
6. Verify tracking works

**Framer wizard (6 steps):**
1. Install tracking scripts (detection + form injection)
2. Verify script installation
3. Build a Pimms-compatible form (if applicable)
4. Configure the Framer webhook (for form tracking)
5. Create a test link
6. Verify tracking works

If you only need Cal.com booking tracking (no Framer forms), the Cal.com wizard alone is sufficient.

## How it works

1. **The Pimms detection script** is installed on your Framer site — when a visitor arrives via a Pimms link, the script detects the click ID
2. **The script auto-injects the tracking ID** into Cal.com embeds on the page via the hidden `pimms_id` field
3. **The visitor books a meeting** through your embedded Cal.com widget — the booking includes the Pimms click ID
4. **Cal.com fires a webhook** to Pimms with the booking data and tracking ID
5. **Pimms attributes the meeting** to the original campaign, visible in your dashboard

The key advantage of this approach is that the detection script handles the handoff between Framer and Cal.com automatically — no manual URL parameter passing needed.

## FAQs

::: faq

### Do I need a paid Framer plan for Cal.com integration?

It depends on how you add the script. Framer's free plan allows custom code in site settings, but a paid plan gives more flexibility. Most users have a paid Framer plan for other features and can add the tracking script easily.

:::

::: faq

### Can I customize the Cal.com booking form appearance?

Yes! Cal.com offers extensive customization options for colors, branding, and form fields. You can customize these in your Cal.com dashboard under Event Types → Advanced → Booking Questions.

:::

::: faq

### Will this work with Framer's CMS?

Yes! You can use Framer's CMS to create dynamic booking pages with different Cal.com links for different team members or services. The detection script works across all pages.

:::

::: faq

### Can I use Pimms short links for the fullscreen calendar method instead of adding ?pimms_id=1?

Yes, but this approach breaks attribution tracking. When you use a Pimms short link directly to Cal.com, all bookings will be attributed to the button click rather than the actual traffic source.

**Use Pimms short links only for:**
- Short-term A/B testing (max 1 month)
- Testing which button design converts better
- Quick setup when you need immediate tracking

For long-term tracking and proper attribution, always use the Cal.com link with `?pimms_id=1` parameter.

:::

::: faq

### Can I track multiple Cal.com calendars on the same Framer site?

Yes! You can embed multiple Cal.com calendars and track them all with Pimms. Each calendar's bookings will be attributed to the specific campaign and page that drove the visitor.

:::
